# Summary

I halved the model for faster printing when testing out different settings (just make sure to allow enough time for cooling).

My best results so far has been with really slow speeds at 0.16mm layer height.
Less wall lines works better than more but only if you allow the bottom skin on top of the walls, see picture from Cura (yellow areas).
